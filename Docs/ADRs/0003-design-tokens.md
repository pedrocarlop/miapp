# ADR 0003: Semantic Design Tokens in DesignSystem

## Status
Accepted

## Decision
Adopt semantic tokens (`ColorTokens`, `SpacingTokens`, `TypographyTokens`, `RadiusTokens`, `ShadowTokens`, `MotionTokens`, `ZIndexTokens`) and reusable DS components (`DSButton`, `DSCard`, `DSText`, `DSChip`, `DSFeedbackBanner`).

## Rationale
- Removes hardcoded UI values from feature code.
- Enables consistent theming and scalable UI composition.
